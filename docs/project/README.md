# Project Tracking

This folder contains MVP-specific tracking documents.

## Files

| File | Purpose | Update Frequency |
|------|---------|------------------|
| [SPEC.md](./SPEC.md) | Project specification | Rarely (scope changes) |
| [MILESTONES.md](./MILESTONES.md) | MVP phases and deliverables | When phases complete |
| [STATUS.md](./STATUS.md) | Current progress | End of each session |
| [DECISIONS.md](./DECISIONS.md) | Decision log | When decisions made |

## Workflow

1. Define specs in SPEC.md at project start
2. Break down into milestones in MILESTONES.md
3. Update STATUS.md at end of each work session (use `/update-status`)
4. Log important decisions in DECISIONS.md as they happen

## Skills

- `/update-status` - Update STATUS.md
- `/new-milestone` - Add milestone with template
- `/create-issues` - Convert milestones to GitHub issues
